# USA₮ Landing (готово для Vercel)

Это статический лендинг (HTML/CSS/JS), подготовленный для деплоя на Vercel.

## Структура

- `index.html` — главная страница
- `css/` — стили
- `js/` — скрипты
- `images/` — изображения
- `media/` — фоновые видео
- `vercel.json` — конфиг для Vercel

## Деплой

1. Создай репозиторий на GitHub и залей туда содержимое папки.
2. На https://vercel.com импортируй репозиторий.
3. В настройках проекта:
   - **Framework Preset**: Other / Static
   - **Build Command**: (пусто)
   - **Output Directory**: `.`

После этого Vercel задеплоит сайт как статический проект.
